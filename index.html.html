Comment ça marche

Simple. Rapide. Fiable.

De la scène terrain au rapport PDF en moins de 5 minutes.

1
Scan QR Code
L'agent scanne le QR affiché par le superviseur. Le formulaire s'ouvre immédiatement sur son téléphone.
2
Remplit le formulaire
Champs intelligents, checklist interactive, ajout de photos depuis la caméra, calcul automatique de la durée.
3
Envoie le rapport
En un clic, le rapport est envoyé via Firebase. Le superviseur le reçoit en temps réel dans son dashboard.
4
Exporte en PDF
Le superviseur visualise, valide et exporte en PDF professionnel A4 bilingue FR/EN aux standards Montage Gold.

ℹ️ Instructions pour les ERT Officers
📱 Comment utiliser le QR Code :
1. Ouvrez l'appareil photo de votre téléphone
2. Scannez le QR Code affiché ci-dessus
3. Une page s'ouvre — remplissez le formulaire
4. Saisissez votre nom, matricule, zone, numéro d'activité
5. Rédigez votre rapport (jusqu'à 5000 mots)
6. Cliquez "Envoyer le rapport"

📌 Note : Le rapport sera visible immédiatement dans cette interface et notifié par email au superviseur.
⚙️ Paramètres du système
🔑 Modifier le mot de passe d'accès

Le mot de passe est stocké localement sur cet appareil. Choisissez un mot de passe sécurisé d'au moins 6 caractères.

Mot de passe par défaut : ERT2025
ℹ️ Note : Le mot de passe est sauvegardé sur cet appareil uniquement. Sur un autre ordinateur, le mot de passe par défaut ERT2025 sera utilisé si aucun nouveau mot de passe n'a été défini sur cet appareil.

Firebase permet de recevoir les rapports des agents en temps réel via le QR Code. Créez un projet gratuit sur firebase.google.com puis collez vos clés ci-dessous.

Guide rapide Firebase :
1. Aller sur firebase.google.com → Créer un projet
2. Activer Realtime Database en mode test
3. Paramètres du projet → Votre application → Copier firebaseConfig
4. Coller les valeurs ci-dessus et sauvegarder
